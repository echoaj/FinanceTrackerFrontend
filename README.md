# Finance Tracker

An CRUD application for users to keep of their expenses.  Uses spring boot for the backend API, React as the frontend, and mongoDB as the database.

![Client View](public/financeTracker.PNG)

## Built With:
#### React.js
#### Java Spring Boot
#### MongoDB
